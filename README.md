# Algebra Practice Tool

A curriculum-aligned algebra practice platform that combines randomized question generation, immediate student feedback, and teacher-facing analytics.

## Overview

The Algebra Practice Tool is designed to help students practice mathematics skills while providing teachers with actionable data about student understanding and engagement.

Unlike generalized learning platforms, this project is built around specific curriculum objectives and pacing guides, allowing practice activities to align directly with classroom instruction.

## Current MVP

The initial version focuses on:

* Randomized algebra question generation
* Multiple-choice practice
* Immediate feedback
* Student response tracking
* Basic teacher analytics
